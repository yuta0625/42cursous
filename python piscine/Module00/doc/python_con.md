if __name__ == '__main__':
   関数名()

このプログラムファイル（.pyファイル）を 「カレーのレシピ」 だと思ってください。

    直接実行する（python file.py）

        これは、あなたがキッチンに立って「よし、今夜はカレーを作るぞ！」と料理を始める状態です。

        この時、レシピの最後に「さあ、調理開始！」と書いてあれば、実際に作り始めますよね。これが if __name__ == '__main__': の中身が実行される状態です。

    インポートする（import file）

        これは、別の人があなたのレシピを参考資料として読む状態です。

        ただレシピを読んでいるだけなのに、勝手に調理が始まったら困りますよね？

        if __name__ == '__main__': があると、別のファイルから読み込まれた（インポートされた）時は、「調理開始！」の合図を無視して、ただの情報の塊（部品）として振る舞います。

2. 仕組みの解説

Pythonには、ファイルの裏側に**名札（__name__）**がついています。

    直接実行した時

        Pythonは、そのファイルの名札を特別に '__main__'（主役） に書き換えます。

        「あ、名札が '__main__' だ！ じゃあこの中のコードを実行しよう」となります。

    インポートされた時

        名札は ファイル名のまま です（例：hello）。

        if 文の条件（'__main__' かどうか）に当てはまらないので、中のコードは無視されます。

pythonで文字列の先頭文字を大文字にするにはえ、str.capitalize()を使う。
