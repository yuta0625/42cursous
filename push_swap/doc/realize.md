push_swapの概要

双方向循環リストであることを考えそれらの要素をどのような動きにしていくかを考える

・aとbという名前のスタックが2つある。
・開始時：
　　・スタックaには、負の数と正の整数がランダムな個数含まれている
　　・スタックbは空である。
・目標はスタックaの数値を昇順に並べ替えることである。

sa(swap a):スタックaの先頭にある最初の2要素を入れ替える
　　　　　　　要素が一つしかない場合、また要素が存在しない場合はなにもしない

sb(swap b):スタックbの先頭にある最初の２要素を入れ替える
　　　　　　　要素が１つしかない場合、または要素が存在しない場合は何もしない

swap(sa/sb)
先頭２つの値/inedxを入れ替えるのが一番簡単である
・top->valueとtop->next->valueをswap
・top->indexとtop->next->indexをswap

ss : saとsbを同時に行う

pa(push a): bの先頭要素をとり、aの先頭におく
　　　　　　　 bが空の場合、何もしない

pb(push b) : aの先頭要素をとり、bの先頭におく
　　　　　　　　aが空の場合、何もしない

ra(rotate a): スタックaの全要素を１つ上にシフトする
　　　　　　　　　最初の要素が最後の要素になる
              ra: a->top = a->top->next;

rb(totate b): スタックbの全要素を１つ上にシフトする
　　　　　　　　　最初の要素が最後の要素になる。

rr : raとrbを同時に行う

rra(reverse rotate a) : スタックaの全要素を１つ下へシフトする
　　　　　　　　　　　　　　　最後の要素が最初の要素になる
						rra: a->top = a->top->prev
　　　　　　　　
rrb(reverse rotate b) : スタックbの全要素を１つ下へシフトする
　　　　　　　　　　　　　　　最後の要素が最初の要素になる

rrr : rraとrrbを同時に行う
