strlcatの使用
→strlcat()は安全な文字列連結関数である。
処理の目的
destの末尾にsrcを最大size-1文字まで追加し、必ず末尾に'\0'をつけて文字列を終端する
destがもうすでにsize文字
if (d_len == size)
	return (size + s_len);
「バッファが小さすぎて実際は結合できなかったけど、もし十分な余裕があったら、全体の長さはこれぐらい（size + s_len）になってたはずだよ」
という意味。
戻り値は「失敗した長さ」ではない
strlcat()は「何文字コピーした」ではなく「コピーできていれば全体がどのくらいの長さになっていたか」を返す関数である。
while (src[i] && (d_len + i + 1) < size)
{
    dest[d_len + i] = src[i];
    i++;
}
dest[d_len + i] = '\0';
「dest が size より小さい場合、src の文字を 最大で size - d_len - 1 文字までコピーし、最後に必ず '\0' で終端する」
＜strlcatは何を守っているのか＞
常に安全に（終端保証付きで）文字列を結合すること」
なので、
•	dest の既存データを壊さない
•	src をコピーできるだけコピー
•	オーバーフロー防止のため、終端文字の位置 (d_len + i + 1) を常にチェック
•	コピーが終わったら dest[d_len + i] = '\0' で安全に終了
